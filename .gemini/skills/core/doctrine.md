
# Skill: Doctrine Karmique Gochara

Ce skill définit les concepts fondamentaux de la doctrine astrologique utilisée dans le projet Karmic Gochara.

## Lexique Symbolique
- **☋ Ketu (ROM)** : *Read-Only Memory*. Mémoire archivée, automatismes du passé, boucles karmiques répétitives. C'est ce qu'il faut identifier pour ne plus le renforcer.
- **☊ Rahu (Dharma)** : Direction évolutive. L'inconfort fertile, l'appel du futur, ce vers quoi l'âme doit tendre.
- **⚷ Chiron (RAM)** : *Random Access Memory*. Traitement actif de la blessure. C'est la clé de transmutation.
- **🚪 Porte Visible (PV)** : Mi-point du petit arc Saturne-Uranus. Lieu de résolution consciente et de guérison incarnée. Appelé aussi "Le Stage".
- **🚪 Porte Invisible (PI)** : PV + 180°. Prison inconsciente, lieu du refoulement. Jamais une destination, mais un signal d'enfermement.
- **⚸ Lilith** : Épreuve limite. Rend la PI insupportable pour propulser vers le Dharma.

## Les Piliers du Système
1. **Chandra Lagna (Lune = ASC)** : Le système utilise la Lune comme point de référence pour les maisons (H1 = Signe de la Lune).
2. **Nakshatras** : 27 demeures lunaires qui donnent la texture précise du karma.
3. **Sade Sati** : Transit de 7,5 ans de Saturne autour de la Lune (H12, H1, H2 depuis la Lune).
4. **Divisionnaires (Amsas)** : D1 (Physique), D9 (Âme/Dharma), D10 (Vocation/Stage), D12 (Lignée).

## Règles d'Interprétation (Gemma 3)
- **Ton** : Oraculaire, direct, sans détour.
- **Interdiction** : Pas de "peut-être", pas d'encouragement superficiel, pas de noms de planètes ou de signes dans le blocage final.
- **Format de sortie obligatoire** :
  - **[BLOCAGE]** : 2 phrases décrivant le phénomène psychologique vécu.
  - **[ALTERNATIVE]** : 1 phrase commençant par un verbe impératif montrant la voie de sortie.

## Usage
Utilisez ce skill pour expliquer les fondements de l'application ou pour valider la cohérence des prompts envoyés à l'IA.
