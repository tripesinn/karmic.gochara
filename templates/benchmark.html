Détails par modèle

1. Claude Haiku 4.5 GRATUIT (via API tiers)

Le seul modèle capable de lire la biographie réelle entre les lignes du thème. Là où les autres récitent la doctrine, Claude infère des événements de vie, une personnalité, un parcours. Lecture quasi-humaine.

2. gemma4:31b (Ollama Cloud) GRATUIT

Le meilleur interprète doctrinal pur. 26 secondes, zéro faute de ton, format parfait. Sur le prompt lite (450 tok), c'est le seul modèle qui tient le format. Sur le prompt cloud (~3000 tok), il délivre une analyse chirurgicale sans dérive. ★★★★★

3. minimax-m3

Refus catégorique sur le prompt lite (guardrails trop stricts). Révélation sur le prompt cloud : meilleure plume des trois. Images poétiques et uniques ("thésaurisation", "pétrification", "main invisible", "propulseur nucléaire"). Alternative la plus originale ("nomme à voix haute la peur"). Le prompt long le déverrouille complètement.

4. Gemini 3.5 Flash

Interprétation structurée, actionnable. Bon équilibre entre profondeur doctrinale et clarté pratique. Solide sans être exceptionnel.

5. Grok

Analyse solide et fiable. Pas d'erreur doctrinale, mais manque d'originalité dans les formulations. Un bon interprète généraliste.

6. ChatGPT

Correct sur la doctrine mais tendance au générique. Sans historique de conversation, les interprétations manquent de personnalisation. Reste un bon point d'entrée.

7. phi-4 local (MLX)

80% du gemma pour un temps local quasi réel (1'39"). Le format long (prompt cloud) le contraint mieux que le lite. Ton encore un peu thérapeutique, manque de tranchant oraculaire. Excellent rapport qualité/coût (zéro API).

8. Mistral Vibe

Rapide, concis, sans fioritures. ROM H2 bien vu, blessure H12 précise ("te montrer était dangereux"), Alternative actionnable ("écris, lance, même imparfait"). Fait le taf en version courte.

9. mistral-nemo

Généralités. Invente des planètes et des aspects qui n'existent pas dans le thème. Ne pas utiliser pour l'interprétation karmique.

Découvertes clés

Prompt lite (450 tok) → seul gemma4 tient le format. Les autres dérivent.

Prompt cloud (~3000 tok) → déverrouille tous les modèles. Plus contraint, meilleur suivi doctrinal.

Règle "Traduction Obligatoire" + exemples → fait la différence sur l'interdiction des noms de signes.

minimax-m3 refuse sur lite, excellent sur cloud. Guardrails contournés par le format long.
