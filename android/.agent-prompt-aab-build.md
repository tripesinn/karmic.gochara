Si une vérif échoue → STOP, remonte l'erreur exact, ne continue pas.

### 2. Incrémenter version

Dans `app/build.gradle`, modifier `defaultConfig` :
```groovy
versionCode 14
versionName "1.4"
```

### 3. Clean

```bash
cd ~/karmic.gochara/android
./gradlew clean
```

### 4. Build AAB release signé

```bash
./gradlew bundleRelease
```

Durée typique : 3–8 min. Si > 15 min, STOP et vérifie `gradle.properties` (daemon, mémoire).

### 5. Vérifier la signature

```bash
AAB=app/build/outputs/bundle/release/app-release.aab
ls -la $AAB

# Vérifier que c'est bien signé avec le bon keystore
jarsigner -verify -verbose -certs $AAB 2>&1 | grep -A 1 "X.509"
# Doit afficher CN = "Jérôme" ou équivalent du keystore karmic-key
```

### 6. Vérifier la cohérence

```bash
# Bundle ID
$ANDROID_HOME/build-tools/*/aapt dump badging $AAB 2>/dev/null | grep -E "package:|versionName|versionCode|application-label"
# Attendu :
#   package: name='com.karmicgochara.app'
#   versionName='1.4'
#   versionCode='14'
#   application-label: 'Karmic Gochara'
```

### 7. Livrer

Donne-moi :
- ✅/❌ statut de chaque étape
- Chemin absolu du `.aab` final
- Taille du fichier (Ko)
- Output des 2 commandes de vérif (signature + aapt)
- Si erreur : stacktrace complet + la ligne Gradle fautive

## Pièges connus (lit AVANT)

| Piège | Fix |
|---|---|
| Keystore pas trouvé (chemin Google Drive) | Vérifier que Google Drive desktop sync a monté le fichier. Si non : `ls -la` du path doit montrer le fichier. Sinon STOP, demande à l'utilisateur de lancer Google Drive. |
| `Execution failed for task ':app:bundleRelease'` | Probablement versionCode déjà utilisé (14 existe en Play) → incrémenter à 15. Demande à l'utilisateur. |
| `SDK location not found` | Exporter `ANDROID_HOME=/Users/jero87/Library/Android/sdk` |
| `JAVA_HOME is set to an invalid directory` | `export JAVA_HOME=$(/usr/libexec/java_home -v 17)` |
| `Out of memory` Gradle | `gradle.properties` → `org.gradle.jvmargs=-Xmx4g` |
| Permission denied sur keystore | `chmod 600 "/Users/jero87/Library/CloudStorage/GoogleDrive-jerome@jeromemalige.fr/Mon Drive/karmic-key"` |
| Signature mismatch (autre keystore) | Si tu vois un CN différent de "Jérôme" → STOP, ne pas uploader. Risque de perte d'accès Play Store. |
